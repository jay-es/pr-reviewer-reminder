# pr-reviewer-reminder

A reminder tool to prevent missing reviewer assignment in Pull Requests.  
https://pr-reviewer-reminder.vercel.app/

## made with

- Vue.js (scaffolded by create-vue)
- VueUse
- Tailwind CSS + daisyUI
- worker-timers
